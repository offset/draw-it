Detailed Description

The Play class.

This class acts as a wrapper for the detection and game functionalities of the program. It calls each in a seperate function. The class is implemented with the singleton pattern, meaning that the classes functions must be called via Play::getInstance()->function.

Parameters

fileNameLevel[in],:how should the generated level file be named
minLength[in],:minimal length for LineFinder class
minGap[in],:minimal gap for LineFinder class
minVote[in],:minimal amount of votes for LineFinder class
skelThreshold,:minimal threshold for the generation of the skeleton of the image
cannyThreshold1[in],:minimal threshold for contour detection with the canny-edge filter.
cannyThreshold2[in],:maximum threshold for contour detection with the canny-edge filter.
cannyApertureSize[in],:Aperture Size for the canny-edge detector. If set higher it should remove some noise, but due to skeleton this is unneccessary.
l2Gradient[in],:If set to true, the calculation is more demanding, but also yields better results.
